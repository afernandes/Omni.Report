# Keyboard Shortcuts — Report Designer

Atalhos seguem a convenção do macOS (`⌘` Cmd, `⌥` Option, `⇧` Shift). No Windows/Linux, troque `⌘` por `Ctrl` e `⌥` por `Alt`.

## Convenções

- **`⌘` vs `Ctrl`**: o app detecta plataforma e renderiza o símbolo correto na UI (palette, tooltips, status bar).
- **Modificador único antes da letra**: `⌘D`, não `D⌘`. Toda kbd na UI segue essa ordem.
- **`Esc`** sempre fecha o overlay/contexto mais externo. Pressionado em sequência: edit text → seleção → painel → tudo limpo.
- **Conflito com browser**: `⌘N`, `⌘O`, `⌘W` são interceptados (preventDefault). Em browsers que bloqueiam, mostra toast: "Use ⌘⇧N etc."
